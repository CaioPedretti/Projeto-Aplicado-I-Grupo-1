## 📌 Objetivo
Este projeto tem como objetivo analisar dados da plataforma Spotify, com foco na identificação de padrões de comportamento dos usuários, 
preferências musicais e tendências de consumo. 
A partir da análise do dataset, busca-se gerar insights que possam auxiliar na compreensão do mercado musical e no uso estratégico dos dados.

## 🏢 Organização
Empresa: Spotify

## 📊 Dataset
Dataset contendo informações de músicas do Spotify, com variáveis como nome da faixa, artista, popularidade, dançabilidade (danceability), 
energia (energy), tempo (tempo) e duração. Esses dados permitem analisar padrões de comportamento dos usuários, identificar preferências musicais e compreender tendências de consumo no mercado digital.
